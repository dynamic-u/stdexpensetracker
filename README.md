Personal Finance Tracker with Smart Insights
Overview:  
A console-based Python application that helps users log daily expenses and incomes, view summaries, and gain smart insights into their spending habits. The project demonstrates core Python concepts like variables, lists/dictionaries, file handling, functions, exception handling, and object-oriented programming.

Features:

Add, delete, and view transactions (expenses/incomes).

Save records persistently in a text file (expenses.txt).

Summarize expenses by category and calculate total income/expenses.

Handle invalid inputs gracefully with exceptions.

Object-oriented design with Transaction and Budget classes.

Real-world Value:  
Budgeting is essential for everyone. This project showcases practical problem-solving skills and is recruiter-friendly because it addresses a universal need.
